# 29 - Flask Mini（最小のFlaskアプリケーション）

この章では、Pythonの軽量なWebフレームワークである **Flask** を使って、最小のWebアプリケーションを作成する方法を学びます。
Flaskを使うと、数行のコードでWebサーバーを立ち上げ、特定のURLにアクセスがあったときに何を実行するかを定義できます。

---

## 準備

この問題を解くには `Flask` ライブラリが必要です。
ターミナルで以下のコマンドを実行して、ライブラリをインストールしてください。
（`27-venv-requirements` のように、このディレクトリで仮想環境を作成してからインストールすることを推奨します）

```bash
pip install Flask
```

---

## Q1: 2つのルートを持つWebアプリケーションの作成

この問題では、2つの異なるURL（ルート）に応答を返す、基本的なFlaskアプリケーションを作成します。

### 要件

1.  `main.py` という名前でファイルを作成します。
2.  Flaskアプリケーションのインスタンスを作成します。
3.  以下の2つのルートを定義してください。
    -   ルートURL (`/`) へのGETリクエストに対して、文字列 `"Hello, Flask!"` を返す。
    -   `/about` というURLへのGETリクエストに対して、文字列 `"This is the about page."` を返す。

---

## アプリケーションの実行方法

1.  ターミナルで、`main.py` があるディレクトリに移動します。
2.  以下のコマンドを実行して、Flaskがアプリケーションを見つけられるように環境変数を設定します。
    ```bash
    export FLASK_APP=main.py
    ```
3.  以下のコマンドで開発サーバーを起動します。
    ```bash
    flask run
    ```
4.  サーバーが起動したら、Webブラウザや `curl` コマンドで `http://127.0.0.1:5000/` や `http://127.0.0.1:5000/about` にアクセスして動作を確認できます。

---

## 正誤チェック方法

```bash
bash check.sh
```
(`check.sh` は、バックグラウンドでFlaskサーバーを起動し、`curl`でアクセスしてレスポンスをテストします)
