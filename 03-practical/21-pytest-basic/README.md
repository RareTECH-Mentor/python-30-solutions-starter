# 21: Pytestの基本

この課題では、Pythonで広く使われているテストフレームワーク `pytest` の基本的な使い方を学びます。

## 概要

`main.py` には、テスト対象となるいくつかの簡単な関数が含まれています。
あなたのタスクは、これらの関数が正しく動作することを確認するためのテストコードを `test_main.py` に記述することです。

## `pytest` のルール

- テストファイルは `test_*.py` または `*_test.py` という名前にします。
- テスト関数は `test_` というプレフィックスで始めます。
- テスト関数内では、`assert` 文を使って期待する結果と実際の値が一致するかを検証します。

## 仮想環境のセットアップと pytest のインストール

プロジェクトごとに依存関係を管理するために、仮想環境を作成することをお勧めします。

1. **仮想環境の作成:**
   ```bash
   python3 -m venv venv
   ```
   (これにより `venv` という名前のディレクトリが作成されます)

2. **仮想環境のアクティベート:**
   - macOS / Linux:
     ```bash
     source venv/bin/activate
     ```
   - Windows:
     ```bash
     venv\Scripts\activate
     ```
   アクティベートされると、プロンプトの先頭に `(venv)` と表示されます。

3. **`pytest` のインストール:**
   仮想環境がアクティベートされた状態で、`pytest` をインストールします。
   ```bash
   pip install pytest
   ```

## テストの実行

1. `main.py` の `add()` 関数と `subtract()` 関数の動作をテストするテスト関数が `test_main.py` に記述されていることを確認してください。
2. ターミナルで `pytest` コマンドを実行し、すべてのテストがパスすることを確認してください。
   ```bash
   pytest
   ```
3. 仮想環境を終了する場合は、`deactivate` コマンドを実行します。

## `check.sh`

このディレクトリの `check.sh` を実行すると、`pytest` が実行され、テストの結果が表示されます。すべてのテストがパスすれば、課題は完了です。
