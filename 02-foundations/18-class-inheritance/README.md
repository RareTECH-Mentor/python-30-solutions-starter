# 18 - Class Inheritance（クラスの継承）

Class Inheritance（クラスの継承）

この章では、**クラスの継承**について学びます。
継承を使うことで、既存のクラスを拡張して新しいクラスを作成できます。

---

## Q01: Base Class

この問題では、基底クラス（親クラス）を定義する方法を学びます。
基底クラスは、他のクラスに継承される基本的なクラスです。

### 要件

- `Animal`クラスを定義します
- `__init__`メソッドで名前を受け取ります
- `speak`メソッドを定義します
- インスタンスを作成してメソッドを呼び出します

```
Dog makes a sound
```

### 注意

- クラス名は大文字で始めます
- `__init__`メソッドはコンストラクタです
- `self`はインスタンス自身を表します

---

## Q02: Inheritance

この問題では、クラスを継承する方法を学びます。
継承を使うことで、基底クラスの機能を引き継ぎながら、新しい機能を追加できます。

### 要件

- `Animal`クラスを継承して`Dog`クラスを作成します
- `speak`メソッドをオーバーライドします
- インスタンスを作成してメソッドを呼び出します

```
Buddy barks
```

### 注意

- 継承は `class 子クラス(親クラス):` の形式で記述します
- メソッドをオーバーライドすることで、親クラスのメソッドを上書きできます

---

## Q03: Multiple Inheritance

この問題では、継承クラスに複数のメソッドを追加する方法を学びます。
継承クラスでは、親クラスのメソッドをオーバーライドしたり、新しいメソッドを追加したりできます。

### 要件

- `Animal`クラスを継承して`Cat`クラスを作成します
- `speak`メソッドをオーバーライドします
- 新しい`purr`メソッドを追加します
- インスタンスを作成して両方のメソッドを呼び出します

```
Whiskers meows
Whiskers purrs
```

### 注意

- 継承クラスでは、親クラスにはない新しいメソッドを追加できます
- 複数のメソッドを定義して、それぞれの機能を実装します
