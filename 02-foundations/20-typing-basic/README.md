# 20 - Typing Basic（型ヒントの基本）

Typing Basic（型ヒントの基本）

この章では、**型ヒント（type hints）**について学びます。
型ヒントを使うことで、コードの可読性と保守性を向上させることができます。

---

## Q01: Type Hints for Variables

この問題では、変数に型ヒントを付ける方法を学びます。
変数の型を明示することで、コードがより読みやすくなります。

### 要件

- 変数に型ヒントを付けます
- 文字列型（`str`）と整数型（`int`）の変数を作成します
- 変数を使って文字列を表示します

```
Alice is 20 years old
```

### 注意

- 型ヒントの構文は `変数名: 型 = 値` です
- `str`は文字列型、`int`は整数型を表します

---

## Q02: Type Hints for Functions

この問題では、関数に型ヒントを付ける方法を学びます。
引数と戻り値の型を指定することで、関数の使い方が明確になります。

### 要件

- 関数の引数に型ヒントを付けます
- 関数の戻り値に型ヒントを付けます
- 関数を呼び出して結果を表示します

```
8
```

### 注意

- 引数の型ヒントは `引数名: 型` の形式です
- 戻り値の型ヒントは `-> 型` の形式です
- この問題では、2つの整数を受け取り、整数を返す関数を定義します

---

## Q03: Type Hints for Lists

この問題では、リストに型ヒントを付ける方法を学びます。
`typing`モジュールの`List`を使って、リストの要素の型を指定します。

### 要件

- `typing`モジュールから`List`をインポートします
- 整数のリストに型ヒントを付けます
- リストの要素の合計を計算して表示します

```
15
```

### 注意

- `List[要素の型]`でリストの型を指定します
- `from typing import List`でインポートします
- この問題では、整数のリスト（`List[int]`）を作成します
