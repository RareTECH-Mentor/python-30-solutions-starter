# 16 - Datetime Basic（日付と時刻の基礎）

この章では、Pythonの標準ライブラリ `datetime` を使って、**日付と時刻**を扱う基本的な方法を学びます。
`datetime`モジュールには、日付や時刻を表現するためのクラス（`datetime`, `date`, `time`, `timedelta`など）が含まれています。

---

## Q1: `datetime`オブジェクトのフォーマット指定

指定された`datetime`オブジェクトを、特定のフォーマットの文字列に変換する方法を学びます。

### 要件

-   `main.py` に、`datetime`モジュールを使って `2023年4月1日 15時30分00秒` を表す`datetime`オブジェクト `dt_obj` を作成します。
-   `strftime()` メソッドを使って、`dt_obj` を `"2023-04-01 15:30:00"` というフォーマットの文字列に変換してください。
-   変換した文字列を標準出力に表示してください。

### ヒント

-   `strftime()` のフォーマットコードは `%Y` (年), `%m` (月), `%d` (日), `%H` (時), `%M` (分), `%S` (秒) などがあります。

### 期待される出力

```
2023-04-01 15:30:00
```

---

## Q2: `timedelta` を使った日付の計算

`timedelta` オブジェクトを使って、特定の日時から一定期間後の日時を計算する方法を学びます。

### 要件

-   Q1で作成した `dt_obj` (`2023年4月1日 15時30分00秒`) を使います。
-   `timedelta` を使って、`dt_obj` の **7日後** の日時を計算してください。
-   計算結果の`datetime`オブジェクトを、Q1と同じ `"YYYY-MM-DD HH:MM:SS"` のフォーマットの文字列に変換して、標準出力に表示してください。

### 期待される出力

```
2023-04-08 15:30:00
```
