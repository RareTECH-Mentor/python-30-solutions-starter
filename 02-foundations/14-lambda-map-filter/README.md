# 14 - Lambda Map Filter（ラムダ、マップ、フィルター）

Lambda Map Filter（ラムダ、マップ、フィルター）

この章では、**lambda式、map()関数、filter()関数**について学びます。
これらは関数型プログラミングの基本的な要素です。

---

## Q01: Lambda Function

この問題では、**lambda式**を使って無名関数を定義する方法を学びます。
lambda式は、短い関数を簡潔に定義するための構文です。

### 要件

- lambda式を使って関数を定義します
- 定義した関数を呼び出します
- 結果を表示します

```
25
```

### 注意

- lambda式の構文は `lambda 引数: 式` です
- この問題では、数値を2乗する関数を定義します

---

## Q02: Map Function

この問題では、**map()関数**を使ってリストの各要素を変換する方法を学びます。
map()関数は、関数とイテラブルを受け取り、各要素に関数を適用した結果を返します。

### 要件

- 数値のリストを作成します
- map()関数とlambda式を使って各要素を2乗します
- 結果をリストに変換して表示します

```
[1, 4, 9, 16, 25]
```

### 注意

- map()関数の結果はmapオブジェクトなので、`list()`でリストに変換します
- lambda式と組み合わせて使います

---

## Q03: Filter Function

この問題では、**filter()関数**を使って条件に合う要素を抽出する方法を学びます。
filter()関数は、関数とイテラブルを受け取り、条件に合う要素だけを返します。

### 要件

- 数値のリストを作成します
- filter()関数とlambda式を使って偶数だけを抽出します
- 結果をリストに変換して表示します

```
[2, 4, 6, 8, 10]
```

### 注意

- filter()関数の結果はfilterオブジェクトなので、`list()`でリストに変換します
- lambda式で条件を指定します（この場合は `x % 2 == 0` で偶数を判定）
