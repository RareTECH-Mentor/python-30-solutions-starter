## Q19: DataClass Basic

この問題では、データクラスに関する問題を解いていきましょう。
データクラスとは、「データがメインで、初期化や表示をPythonに任せられるクラス」のことです。
データクラスを使用することで、データを持つためのクラスを簡単に書くことができます。

---

### 事前準備

データクラスの使用には、
標準ライブラリ`dataclasses`から`dataclass`デコレータのインポートが必要です。

以下を`models.py`に書いて、models.py内でデータクラスが使用できる状態にしましょう。

```
from dataclasses import dataclass
```

### 問１：データクラスを定義しよう

`Person`というデータクラスを`models.py`内に作成してください。

**条件**

- 属性
  - name：文字列
  - age：整数

### 問２：インスタンスを作成して表示しよう

問題1で作成した`Person`クラスを使って、次の人物を表すインスタンスを作成してください。

- name：Alice
- age：20

また、作成したインスタンスを`print`してください。

**期待する出力値**

```
Person(name='Alice', age=20)
```

### 問５：データクラスにメソッドを追加する

5-1.
以下のメソッドを`Book`クラスに追加してください。
このメソッドは割引率を引数に取り、割引後の金額を戻り値として返します。

**メソッドの仕様**

- メソッド名：discount
- 引数：rate（整数）
  - 割引率を表す（例：20 → 20%引き）
- 戻り値：割引後の金額（整数）

**割引の計算方法**

```
割引後の金額 ＝
price × (100 − rate) // 100
```

注）小数が出る場合は、小数点以下を切り捨てることとする

`discount`メソッドを使用して表示してください。

- Bookインスタンス３
  - title：AWS実践
  - price：1000

**期待する出力値**
