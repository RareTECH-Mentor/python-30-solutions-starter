## Q13: Exceptions Basic

この問題では、例外処理に関する問題を解いていきましょう。
例外処理とは、プログラムの実行中に起きるエラー（例外）を受け取り、
プログラムが止まらないように対処する仕組みです。

---

### 問１：例外処理をしてみよう

次の仕様を満たすPythonプログラムを作成してください。

**仕様**

- ユーザーに整数を入力してもらう
- その際、ユーザーが何を入力するべきか分かるように「整数を入力してください：」と表示する
- 入力された値は`整数型`に変換する
- 変換できた場合は「入力された数値は〇〇です。」と表示する
- 整数に変換できない入力がされた場合は「入力された値を整数に変換できません。整数を入力してください。」と表示する

**条件**

- 「整数を入力してください:」のコロン（：）は全角であること
- 例外処理を行うこと
- プログラムがエラーで終了しないようにすること

**実行例**

正しい入力の場合

```
整数を入力してください：10
入力された数値は10です。
```

誤った入力の場合

```
整数を入力してください：abc
入力された値を整数に変換できません。整数を入力してください。
```

---

### 問２：正しい入力まで繰り返す処理をしてみよう

問１と同様にユーザーに整数を入力してもらい、その数値を整数型に変換します。
もし例外が発生した場合は適切な数値が入力されるまで処理を繰り返してみましょう。

**条件**

- 「整数を入力してください:」のコロン（：）は全角であること
- 例外処理を行うこと
- 例外が発生した場合は再度ユーザーに整数を入力するように促し、適切に処理が終了するまで繰り返すこと
