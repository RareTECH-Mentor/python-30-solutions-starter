# 12 - File Read Write（ファイルの読み書き）

File Read Write（ファイルの読み書き）

この章では、**ファイルの読み書き**について学びます。
Pythonでは、`open()`関数と`with`文を使ってファイルを操作します。

---

## Q01: File Write

この問題では、ファイルに文字列を書き込む方法を学びます。
`open()`関数と`write()`メソッドを使ってファイルに書き込んでみましょう。

### 要件

- `open()`関数を使ってファイルを開きます
- `write()`メソッドで文字列を書き込みます
- `with`文を使ってファイルを安全に閉じます

```
（ファイル output.txt に "Hello, Python!" が書き込まれます）
```

### 注意

- ファイルモードは "w"（書き込み）を使用します
- `encoding="utf-8"` を指定します
- `with`文を使うことで、ファイルが自動的に閉じられます

---

## Q02: File Read

この問題では、ファイルから内容を読み込む方法を学びます。
`read()`メソッドを使ってファイルの内容を読み込んでみましょう。

### 要件

- `open()`関数を使ってファイルを開きます
- `read()`メソッドでファイルの内容を読み込みます
- 読み込んだ内容を表示します

```
Hello, Python!
```

### 注意

- ファイルモードは "r"（読み込み）を使用します
- `encoding="utf-8"` を指定します
- Q01で作成した output.txt ファイルを読み込みます

---

## Q03: File Append

この問題では、ファイルに追記する方法を学びます。
`open()`関数のモード "a"（追記）を使って、既存のファイルに新しい内容を追加してみましょう。

### 要件

- `open()`関数のモード "a" を使ってファイルを開きます
- `write()`メソッドで新しい行を追記します
- 追記後のファイルの内容を読み込んで表示します

```
Hello, Python!
This is a new line.
```

### 注意

- ファイルモードは "a"（追記）を使用します
- 追記後、ファイルを読み込んで内容を確認します
- 改行文字 "\n" を使って新しい行を追加します
