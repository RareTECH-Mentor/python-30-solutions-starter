## Q17: Class Basic

この問題では、クラスに関する問題を解いていきましょう。
クラスとは、「ものを作るための設計図」です。
クラスからインスタンス（実際に使うもの）を作成することで、以下の利点があります。

1. 同じ処理をまとめられて、コードが読みやすくなる
2. データと動きをセットで管理できる
3. 同じ種類のものを、何個でも安全に作れる
4. 変更に強くなる

---

### 問１：クラスを作成してみよう

`Person.py`内に以下の要件を満たす「Person」クラスを作成してください。
**要件**

- クラス名：Person
- 属性：
  - name（名前）
  - age（年齢）
- メソッド：
  - say_hello()：`私の名前は〇〇です`と表示する
  - introduce()：`私の年齢は〇〇歳です` と表示する

---

### 問２：インスタンスを作成してみよう

問題1の `Person`クラスからインスタンスを作成し、
`say_hello`メソッドと`introduce`メソッドを実行してください。

**say_helloメソッド実行例**（インスタンスのnameが次郎の場合）

```
私の名前は次郎です
```

**introduceメソッド実行例**（インスタンスのageが20の場合）

```
私の年齢は20歳です
```

**条件**

- メソッドの実行はインスタンスを作成して行うこと

---

### 問３：戻り値のあるメソッドを作成しよう

`Rectangle.py`内に以下の要件を満たす`Rectangle`クラスを作成してください。

**要件**

- 属性：
  - width
  - height
- メソッド：
  - area()：widthとheightから面積を計算し、その値をreturnする

### 問４：戻り値を出力しよう

問題３の `Rectangle`クラスからインスタンスを作成し、
`area`メソッドを実行し、その戻り値を以下のように出力してください。

**出力結果**（インスタンスのwidthが3, heightが4の場合）

```
辺の長さが3と4の四角の面積は12です
```

**条件**

- メソッドの実行はインスタンスを作成して行うこと
- 表示する文字列「辺の長さが3と4の四角の面積は12です」内の3・4・12はインスタンスに応じて動的に数字が変わるようにすること
