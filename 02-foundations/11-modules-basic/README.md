## Q11: Modules Basic

この問題では、モジュールに関する問題を解いていきましょう。
モジュールとは「機能ごとにまとめた“1つの .py ファイル”」のことです。
コードが長くなるのを防いだり、役割ごとに整理するために使用されます。
モジュールは以下の３パターンです。

1. 自分で定義するモジュール（自分で作った`.py`ファイル）
2. 標準ライブラリのモジュール（Pythonに最初から入っているためインストールは不要）
3. 外部ライブラリのモジュール（第三者が作成したもので、pipでインストールして使用する）

---

### 問１：標準ライブラリをimportしよう

math モジュールを import して、
25 の平方根を表示するプログラムを書きましょう。

**要件**

- `math`モジュールを全てimportすること

**出力結果**

```
25の平方根は5.0です
```

---

### 問２：未達成の学習科目を表示しよう

`math`モジュールから `pi` をimportし、
円周率を表示してください。

**要件**

- `math.pi`と記述しないこと

**出力結果**

```
円周率は3.141592653589793です
```

---

### 問３： as を使って別名をつけてみよう

`random`モジュールを `r` という名前でimportし、
1〜6 のランダムな整数を1つ表示するプログラムを書きましょう。

**要件**

- 数値は整数であること
- 整数は1〜6の範囲であること

---

### 問４：自作モジュールを作成しよう

`main.py`から`greet.py`をimportして、
hello() を呼び出すコードを書きましょう。

**処理内容**

- `greet.py`の中に`hello()`という関数を作成する
- `hello`関数は引数に名前を受け取り、「Hello, 〇〇（引数の名前）!」を表示する
- `main.py`で`greet.py`をimportして`hello()`を実行する

**要件**

- `main.py`内に`hello()`を定義してはならない

**出力結果**（引数が`Python`の場合）

```
Hello Python!
```
