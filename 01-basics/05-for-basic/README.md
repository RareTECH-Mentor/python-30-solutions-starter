## Q05: For Basic

この問題では、For 文 を使って繰り返し処理を行なっていきます。

---

## 要件

- 「Hello Python」を５回出力する。
- 出力時に、各行の最初に行番号をつける。（例）1：, 2：, ...
- コロン(：)は全角で入力すること。
- 必ず For 文を使用すること。

**出力結果**

```
1：Hello Python
2：Hello Python
3：Hello Python
4：Hello Python
5：Hello Python

```

---

## 変更するファイル

- main.py

---

## 正誤チェック方法

```bash
python main.py
bash check.sh
```
