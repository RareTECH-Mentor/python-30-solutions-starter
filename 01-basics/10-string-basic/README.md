# 10 - String Basic（文字列）

String Basic（文字列）

この章では、**文字列（strings）** について学びます。
文字列とは、**文字のシーケンスを表現するデータ型**です。

---

## Q01: String Basic

この問題では、Pythonの基本要素である **文字列の使い方** を学びます。
文字列を作成し、その内容を表示してみましょう。

### 要件

- Pythonを使ってプログラムを書く
- 文字列を作成する
- 文字列を表示してみましょう。

```
Hello
```

### 注意

- 文字列を使うこと
- 文字列を正しく表示すること

---

## Q02: String Concatenation

この問題では、文字列の連結方法を学びます。
複数の文字列を連結してみましょう。

### 要件

- 文字列を作成する
- 文字列を連結する
- 連結した文字列を表示してみましょう。

```
Hello World
```

### 注意

- 文字列の連結演算子を使うこと
- スペースを適切に入れること
