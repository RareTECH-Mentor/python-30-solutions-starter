# 10 - String Basic（文字列）

String Basic（文字列）

この章では、**文字列（strings）** について学びます。
文字列とは、**文字のシーケンスを表現するデータ型**です。
この章では特に、**split()メソッドとjoin()メソッド**の使い方を学びます。

---

## Q01: String Split

この問題では、文字列を分割する **split()メソッド** の使い方を学びます。
カンマ区切りの文字列を分割してみましょう。

### 要件

- 文字列を作成します
- split()メソッドを使って文字列を分割します
- 分割結果を表示します

```
['apple', 'banana', 'orange']
```

### 注意

- split()メソッドの引数に区切り文字を指定します
- カンマ "," で分割します

---

## Q02: String Join

この問題では、リストを結合する **join()メソッド** の使い方を学びます。
複数の文字列をスペースで結合してみましょう。

### 要件

- 文字列のリストを作成します
- join()メソッドを使ってリストを結合します
- 結合結果を表示します

```
Hello World Python
```

### 注意

- join()メソッドは文字列のメソッドです
- 結合する区切り文字を指定します（この場合はスペース " "）

---

## Q03: Split and Join

この問題では、split()とjoin()を組み合わせて使う方法を学びます。
文字列を分割して、別の区切り文字で結合してみましょう。

### 要件

- 文字列を作成します
- split()メソッドで文字列を分割します
- join()メソッドで別の区切り文字を使って結合します
- 結果を表示します

```
one_two_three
```

### 注意

- split()で分割した後、join()で結合します
- 元の区切り文字 "-" から、新しい区切り文字 "_" に変更します
