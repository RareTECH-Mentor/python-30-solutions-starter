# 02 - Variables（変数）

Variables（変数）

この章では、**変数（variables）** について学びます。
変数とは、**データに名前をつけて保存する仕組み**です。

---

## Q01: Variables Basic

この問題では、Pythonの基本要素である **変数の使い方** を学びます。
数値を変数に代入しみましょう。

### 要件

- Pythonを使ってプログラムを書く
- 数値を変数に代入する
- 変数の値を使ってみましょう。

```
My age is 20
```

### 注意

- 数字は **変数を使ってみましょう。**


---

## Q02: String Variable

この問題では、文字列を変数に代入する方法を学びます。
名前を表す文字列を変数に入れましょう。


### 要件

- 文字列を変数に代入する
- 変数を使ってみましょう。

```
My name is Alice
```

### 注意

- 文字列は ダブルクォート " で囲むこと


---

## Q03: Multiple Variables

この問題では、2つの変数を使って表示する方法を学びます。
数値と文字列、それぞれを変数に代入し、
それらを組み合わせて表示してみましょう。

### 要件

- 文字列用の変数を1つ作る
- 数値用の変数を1つ作る


```
My name is Alice and I am 20 years old
```

### 注意

- すべて 変数を使って表示すること
