## Q04: Type Conversion

この問題では、Python を使ってキーボードからの入力を受け付け、
入力されたデータを加工した後に出力していきます。

---

## 要件

- キーボードから年齢の入力を受け付ける。その際、ユーザーが何を入力するか分かるように「年齢を入力してください：」と表示する。
- ユーザーが入力した年齢から数え年を計算し、「あなたの数え年は〇〇歳です」と表示する。
- 数え年は年齢＋１で計算するものとする。

入出力例

```
年齢を入力してください：20
あなたの数え年は21歳です
```

---

## 変更するファイル

- main.py

---

## 正誤チェック方法

python main.py
