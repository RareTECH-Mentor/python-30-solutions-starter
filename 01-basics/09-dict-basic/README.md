## Q09: Dictionary Basic

この問題では、辞書型についての問題を解いていきましょう。
辞書型とは、**名前（キー）とそれに対応する値をセットでしまう箱** のことで、
名簿や電話帳のようなイメージです。

---

### 問１：辞書型のデータを定義しよう

以下の要件を満たす辞書型のデータを作成してください。

- 変数名は progress とする
- key は 学習科目名（文字列）を入れる
- value は 達成状況（True / False）を入れる
- 次のデータを全て持つこと
  | 科目 | 達成状況 |
  | ---------- | ---- |
  | Python | 未達成 |
  | HTML | 達成 |
  | CSS | 未達成 |
  | JavaScript | 未達成 |

---

### 問２：未達成の学習科目を表示しよう

2-1. 問１で定義した辞書から、未達成の科目（value が Falseのもの）だけを全て表示する関数 `print_unfinished()` を定義してください。

**関数の仕様**

```
def print_unfinished():
    print("----------------")
    print("未達成の科目:")
    ...
```

**処理内容**

- 出力結果の以下の表示から始まるようにする（こちらは既にmain.py内に記述されているので消さないこと）
  ```
  ----------------
  未達成の科目:
  ```
- 未達成の科目を `- 科目名` の形式で表示する
- 関数内で `print()` を使用し、return は使わない
- 関数の実行結果が以下のようになるようにする

```
----------------
未達成の科目:
- Python
- CSS
- JavaScript
```

2-2. `print_unfinished()` 関数を実行するコードを記述してください。

---

### 問３：学習科目を達成済みにしよう

3-1. 引数で渡された学習科目を 達成済み（True）に変更する関数 `complete_subject(subject)` を定義してください。

**関数の仕様**

```
def complete_subject(subject):
    ...
```

**処理内容**

- 引数 subject（科目） が progress（辞書） に 存在する場合：
  - その科目の value を True に変更する
- 存在しない場合：
  - 何もしない

3-2. 以下のように、定義した関数を実行して、未達成の科目を達成済みに変更するコードを記述してください。

```
complete_subject("Python")
complete_subject("CSS")
```

3-3. 更新された辞書を使って未達成の科目を表示し、辞書の中身が変更されたことを確認しましょう。

**要件**

- 問２で作成した関数を再度呼び出し、未達成の科目が少なくなっていることを確認してください。
