## Q03: Basic I/O

この問題では、Python を使ってキーボードからの入力を受け付け、
入力された値を出力していきます。

---

## 要件

### 問１

- キーボードから名前の入力を受け付ける。その際、ユーザーが何を入力するか分かるように「名前を入力してください：」と表示する。
- ユーザーが入力した名前を使い、「Hello, 〇〇!」と表示する。

入出力例

```
名前を入力してください：Alice
Hello, Alice!
```

### 問２

- キーボードから年齢の入力を受け付ける。その際、ユーザーが何を入力するか分かるように「年齢を入力してください：」と表示する。
- ユーザーが入力した年齢を使い、「あなたの年齢は〇〇歳です」と表示する。

入出力例

```
年齢を入力してください：20
あなたの年齢は20歳です
```
