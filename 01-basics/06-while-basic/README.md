# 06 - While Basic（繰り返し処理）

While Basic（繰り返し処理）

この章では、**while文（while loops）** について学びます。
while文とは、**条件が真の間、処理を繰り返す仕組み**です。

---

## Q01: While Basic

この問題では、Pythonの基本要素である **while文の使い方** を学びます。
条件が真の間、処理を繰り返してみましょう。

### 要件

- Pythonを使ってプログラムを書く
- while文を使って繰り返し処理を実装する
- 条件に応じた処理を繰り返してみましょう。

```
1
2
3
```

### 注意

- while文を使うこと
- 無限ループに注意すること
