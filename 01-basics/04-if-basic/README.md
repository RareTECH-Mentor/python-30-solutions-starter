# 04 - If Basic（条件分岐）

If Basic（条件分岐）

この章では、**条件分岐（if statements）** について学びます。
条件分岐とは、**条件に応じて処理を分ける仕組み**です。

---

## Q01: If Basic

この問題では、Pythonの基本要素である **if文の使い方** を学びます。
条件が真の場合に処理を実行してみましょう。

### 要件

- Pythonを使ってプログラムを書く
- if文を使って条件分岐を実装する
- 条件に応じた処理を実行してみましょう。

```
Adult
```

### 注意

- if文を使うこと
- 条件式を正しく書くこと

## Q02: If-Else

条件が真の場合と偽の場合で処理を分岐してみましょう。

### 要件

- if-else文を使って条件分岐を実装する
- 条件に応じた処理を実行してみましょう。

```

```

または

```

```

### 注意

- if-else文を使うこと
