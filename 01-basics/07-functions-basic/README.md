## Q07: Functions Basic

この問題では、関数を使って特定の処理をまとめていきます。

---

## 要件

### 問１：「Hello Python」を３回表示する関数

- 関数名は print_hello_python とする。
- **関数の中**で「Hello Python」を 3 回出力する。
- 繰り返しには for または while を使用する。
- 定義した関数を 1 回呼び出すこと。

出力結果

```
Hello Python
Hello Python
Hello Python
```

### 問２：名前を受け取って表示する関数

- say_hello という名前の関数を定義する。
- 引数として ユーザーの名前を 1 つ受け取る。
- 引数で受け取った名前を使用し、**関数の中で**「Hello 〇〇(ユーザー名)」を出力する。
- 引数に渡す名前は自由とする。ただし空文字は不可。
- 定義した関数を 1 回呼び出すこと。

出力例（名前が Alice の場合）

```
Hello Alice
```

### 問３：戻り値のある関数

- format_name という名前の関数を定義する。
- 引数として ユーザーの名前を 1 つ受け取る。
- 関数の中では、受け取った名前を使って「Hello 〇〇(ユーザー名)」という文字列を作り、return すること。
- 関数の中で print などを使って出力してはならない。
- 定義した関数を 1 回呼び出し、戻り値を使って出力すること。
- 呼び出す際に渡す名前は自由とする。

出力例（名前が 太郎 の場合）

```
Hello 太郎
```
