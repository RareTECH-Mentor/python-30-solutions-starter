## Q07: Functions Basic

この問題では、関数を使って特定の処理をまとめていきます。

---

### 問１：「Hello Python」を３回表示する関数

「Hello Python」を３回表示する関数を定義・実行してみましょう。

**処理内容**

- 関数名を`print_hello_python`とする。
- **関数の中**で「Hello Python」を 3 回出力する。
- 繰り返しには for または while を使用する。
- 定義した関数を 1 回呼び出すこと。

**出力結果**

```
Hello Python
Hello Python
Hello Python
```

---

### 問２：引数を受け取って表示する関数

引数として名前を受け取り、その名前を使用して`Hello (引数で受け取る名前)`を表示する関数を定義・実行しましょう。

**処理内容**

- 関数名を`say_hello`とする。
- 引数として ユーザーの名前を1つ受け取る。
- 引数で受け取った名前を使用し、**関数の中で**「Hello 〇〇(ユーザー名)」を出力する。
- 引数に渡す名前は自由とする。ただし空文字は不可。
- 定義した関数を 1 回呼び出す。

**出力例**（名前が Alice の場合）

```
Hello Alice
```

---

### 問３：戻り値のある関数

こちらの問題では問題２と同じように引数として名前を受け取りますが、関数内で表示せず戻り値として`Hello (引数で受け取る名前)`を返し、関数の外で表示させます。

**処理内容**

- 関数名を`format_name`とする。
- 引数として ユーザーの名前を1つ受け取る。
- 関数の中では、受け取った名前を使って「Hello 〇〇(ユーザー名)」という文字列を作り、関数内で表示でずにreturn する。
- 関数の中で print などを使って出力してはならない。
- 定義した関数を 1 回呼び出し、戻り値を使って出力する。
- 呼び出す際に渡す名前は自由とする。ただし空文字は不可。

**出力例**（名前が 太郎 の場合）

```
Hello 太郎
```
