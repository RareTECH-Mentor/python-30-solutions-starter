# 08 - List Basic（リスト）

List Basic（リスト）

この章では、**リスト（lists）** について学びます。
リストとは、**複数の値を順番に並べて保存するデータ構造**です。

---

## Q01: List Basic

この問題では、Pythonの基本要素である **リストの使い方** を学びます。
リストを作成し、その要素を表示してみましょう。

### 要件

- Pythonを使ってプログラムを書く
- リストを作成する
- リストの要素を表示してみましょう。

```
[1, 2, 3]
```

### 注意

- リストを使うこと
- リストの要素を正しく表示すること

---

## Q02: List Access

この問題では、リストの要素にアクセスする方法を学びます。
リストの特定の要素を取り出してみましょう。

### 要件

- リストを作成する
- リストの要素にアクセスする
- 特定の要素を表示してみましょう。

```
2
```

### 注意

- インデックスを使って要素にアクセスすること
- インデックスは0から始まること

---

## Q03: List Operations

この問題では、リストの操作について学びます。
リストに要素を追加したり、変更したりしてみましょう。

### 要件

- リストを作成する
- リストに要素を追加する
- リストの要素を変更する
- 変更後のリストを表示してみましょう。

```
[1, 2, 4]
```

### 注意

- リストの操作メソッドを使うこと
- リストを正しく更新すること
